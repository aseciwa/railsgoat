## Getting Started ##
#### With Ruby, Rubygems, Git, and Bundler installed ####

	git clone https://github.com/OWASP/railsgoat.git

	cd railsgoat

	rvm use 1.9.3@railsgoat --create # https://rvm.io/

	bundle

	rake db:setup

	rails s

	open http://0.0.0.0:3000

	Start hacking!!!
